# 💻 Algoritmos e Programação

Repositório dedicado ao armazenamento da minha documentação técnica, notas de estudo e resoluções práticas da disciplina de Algoritmos e Programação do curso de Análise e Desenvolvimento de Sistemas (ADS).

> 🚀 **Metodologia de Estudo:** Este repositório funciona como uma "Wikipédia pessoal". A teoria e os conceitos fundamentais são estruturados em arquivos Markdown, enquanto a prática laboratorial, listas e projetos são desenvolvidos puramente em scripts executáveis.

---

## 🛠️ Tecnologias e Ferramentas

* **Linguagem Base:** Python 3.13 🐍
* **Ambiente de Desenvolvimento:** VS Code 🚀
* **Documentação:** Markdown `(.md)` 📑

---

## 📂 Organização do Repositório

O projeto é dividido em quatro pilares fundamentais para garantir uma consulta rápida e um histórico acadêmico limpo:

* 📖 **`conteudos/`**: Notas de aula e resumos teóricos divididos por módulos conceituais.
* 📝 **`exercicios/`**: Listas de exercícios oficiais aplicadas durante o semestre.
* 🎓 **`avaliacoes/`**: Arquivo histórico de provas oficiais e resoluções de semestres anteriores.
* 🧪 **`extras/`**: Meu laboratório de experimentos, códigos de livros externos e mini-projetos autorais.
